# DeleteLine
Delete Line Xcode Source Editor Extension (删除选中行的Xcode Source Editor 插件)

![Demo](https://github.com/CatchZeng/DeleteLine/blob/master/demo.gif)

## Usage

#### 1.Clone/Download Project

#### 2.Run Project(command+R)

![Run](https://github.com/CatchZeng/DeleteLine/blob/master/run.png)

#### 3.Open a project,such as Test project

![test](https://github.com/CatchZeng/DeleteLine/blob/master/test.png)

#### 4.Click Editor menu, you can see it

![editor](https://github.com/CatchZeng/DeleteLine/blob/master/editor.png)

#### 5.key bindings,then you can trigger it by command + d

![bindings](https://github.com/CatchZeng/DeleteLine/blob/master/bindings.png)
